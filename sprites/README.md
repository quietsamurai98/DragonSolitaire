# Sprite Files
### Cards
File name format: `card_[suit]_[value].png`
* Suit values:
  * `r` - Red cards
  * `g` - Green cards
  * `b` - Blue cards
  * `m` - Miscellaneous cards
* Value... values:
  * Color suit cards:
    * `1-9` - Standard numeric cards
    * `0` - Dragon cards
  * Miscellaneous cards:
    * `0` - Flower card
